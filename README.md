# 打飞机小游戏 🚀

一款基于 Python + Pygame 的经典打飞机小游戏，支持 8 个关卡和 8 种不同类型的敌机。

## 游戏特性

- ✅ 流畅的飞机控制系统（方向键/WASD）
- ✅ 8 种不同属性的敌机类型
- ✅ 完整的关卡系统（8个关卡，难度递增）
- ✅ 背景音乐和音效系统
- ✅ 分数和生命值系统
- ✅ 进度追踪和关卡过渡动画
- ✅ 完整的中文界面支持

## 环境要求

- macOS 10.13+
- Python 3.7+
- 至少 100MB 可用内存

## 安装与运行

### 1. 安装依赖

```bash
pip3 install -r requirements.txt
```

### 2. 运行游戏

```bash
python3 main.py
```

## 游戏操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 向上移动 | ⬆️ 或 W | - |
| 向下移动 | ⬇️ 或 S | - |
| 向左移动 | ⬅️ 或 A | - |
| 向右移动 | ➡️ 或 D | - |
| 射击 | 空格 | 200ms冷却时间 |
| 重新开始 | R | 仅游戏结束时 |
| 退出游戏 | ESC | 任何时候 |

## 游戏规则

1. **目标**: 击败每关的目标敌机数（每关3架），通过所有8个关卡
2. **生命值**: 初始3条生命，被撞击或被子弹击中损失1条
3. **得分**: 不同类型敌机给予不同分数（10-100分）
4. **通关**: 完成所有8关显示"恭喜通关！"
5. **失败**: 生命值归零显示"游戏结束！"

## 技术亮点

1. **中文字体支持**: 使用 macOS 系统字体（苹方黑体）完美支持中文显示
2. **关卡配置系统**: 使用配置文件分离数据和逻辑，便于扩展
3. **敌机类型系统**: 支持多种敌机类型，每种敌机有独特的属性
4. **音效系统**: 完整的背景音乐和音效支持
5. **进度追踪**: 实时显示关卡进度和过渡动画
